Usage:

```jsx static
import SVGIcon from 'components/special/SVGIcon';
```

Basic render:

```html
<SVGIcon src={icon}/>
```

Custom props:

width, height

```html
<SVGIcon src={icon} width={20} height={30}/>
```

size

```html
<SVGIcon src={icon} size={25} />
```

color

```html
<SVGIcon src={icon} color="purple"/>
```
